# PWA Icons

Placez les icônes PWA suivantes dans ce dossier :

- `icon-192x192.png` - Icône 192x192 pixels (maskable)
- `icon-512x512.png` - Icône 512x512 pixels

## Génération des icônes

Vous pouvez utiliser un outil comme [PWA Asset Generator](https://github.com/nickytonline/pwa-asset-generator) ou [RealFaviconGenerator](https://realfavicongenerator.net/) pour générer ces icônes à partir d'un logo SVG.

## Spécifications

- Format: PNG
- Fond: Transparent ou #1B4F72 (primary color)
- Contenu: Logo KAROSSE centré
